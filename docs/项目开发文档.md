# 个人成长记录型博客网站 - 产品需求文档 (PRD)

## 项目概述

一个科幻风格的个人成长记录博客网站，以时间轴为核心，记录个人成就、习惯养成、里程碑事件等生命节点。支持多媒体内容管理，具备独特的管理员登录机制，提供沉浸式的成长轨迹探索体验。

---

## 一、核心功能需求

### 1 时间轴主页

**1.1 时间轴展示**

- 以线性时间轴为主要视觉元素，贯穿页面主体
- 节点沿时间轴排列，节点间距与日期时间差值成正比
- 支持时间轴缩放（鼠标滚轮控制比例系数）与拖动交互
- 时间轴线条采用霓虹色（亮红/粉紫），带科技感光效（微微闪烁/流动效果）

**1.2 节点呈现**

- 节点以几何形状（三角/圆/方）呈现，支持霓虹发光效果
- 节点大小与"级别字段"关联：缩放比例 = (级别数值 - 1) / 3 + 1（默认1.0）
- 节点颜色饱和度与级别数值正相关
- 节点上简洁显示关键信息提示（简短标题/时间）

**1.3 节点详情展示**

- 点击节点后，时间轴自动拖动使该节点位于页面中央
- 从节点处平滑展开详细内容面板（动态过渡效果）
- 详情面板延续赛博朋克风格（霓虹元素、科技感边框）
- 支持多媒体内容展示：文本、数字、图片、音频、视频(mp4)

---

### 2 多媒体内容管理

**2.1 文件处理**

- 图片自动压缩
- 音视频转码，确保兼容性
- 文件存储结构：图片保存原图+缩略图，音视频仅保存原文件

**2.2 文件上传逻辑**


| 场景       | 保存按钮                   | 取消/关闭按钮                                |
| -------- | ---------------------- | -------------------------------------- |
| **新增节点** | 只保留预览列表中的文件，清理未引用的临时文件 | 删除本次会话所有新上传文件（图片原图+缩略图/音视频原文件），清空数据库记录 |
| **编辑节点** | 只保留预览列表中的新文件，清理未引用临时文件 | 恢复到编辑前状态，删除本次新上传的所有文件，原有文件完整保留         |


**2.3 文件清理规则**

- 图片：删除原图 + 缩略图
- 音频/视频：仅删除原文件
- 通过文件名/唯一ID追踪文件

---

### 3 搜索功能

- 支持关键词检索节点标题、内容、标签
- 搜索结果以列表形式展示
- 点击搜索结果跳转至对应节点详情（自动定位时间轴）

---

### 4 管理员界面与特殊登录机制

**4.1 登录入口**

- 访问 `/admin` 路径进入管理员登录页
- 登录页仅有一个密码输入框

**4.2 特殊登录机制**

```text
1. 输入框聚焦时：正常键盘输入（字符只显示在输入框，但不保存到后台）
2. 输入框失焦时：激活键盘监听
   - 键盘输入的字符被后台保存（但输入框不显示字符）
   - 按下Enter时，将保存的字符串与"chi"比对
   - 匹配成功 → 进入管理员模式
   - 匹配失败 → 清空已保存字符，重新记录
```

**4.3 管理员模式功能**

- 跳转回时间轴主页
- 增加节点操作权限：
  - **新增节点**：点击"新增节点"按钮进入编辑页
  - **编辑节点**：先点击"编辑节点"按钮，再点击目标节点
  - **删除节点**：先点击"删除节点"按钮，再点击目标节点（需二次确认）
- 节点编辑页面包含：标题、内容、时间、标签、级别(浮点型)字段
- 编辑页支持多媒体文件上传/预览/删除
- 页面顶部显示"退出管理员模式"按钮

---

### 5 编辑界面交互规范

**5.1 新增节点流程**

```text
点击新增节点按钮 → 进入空白编辑页
    ↓
填写信息/上传文件（文件实时上传至服务器，生成预览）
    ↓
[保存] 按钮：验证信息 → 保存至数据库 → 更新时间轴 → 关闭编辑页
[取消]/[×] 按钮：删除本次上传的所有文件 → 关闭编辑页（无数据库变更）
```

**5.2 编辑节点流程**

```text
点击编辑节点按钮 → 点击目标节点 → 进入预填信息的编辑页
    ↓
修改信息/上传新文件（新文件实时上传）
    ↓
[保存] 按钮：只保留预览列表中的文件 → 清理未引用临时文件 → 更新数据库 → 更新时间轴 → 关闭编辑页
[取消]/[×] 按钮：删除本次新上传的所有文件 → 恢复到编辑前状态 → 关闭编辑页（无数据库变更）
```

**5.3 删除节点流程**

```text
点击删除节点按钮 → 点击目标节点 → 弹出确认提示
    ↓
[确认] 删除：删除节点 → 删除关联的多媒体文件 → 更新时间轴
[取消]：返回正常浏览状态
```

### 6 沉浸式环境效果

**6.1 背景音乐**

- 网页添加循环播放的背景音乐

**6.2 天气/场景更迭** 随着时间轴往前推进，模拟从起点穿过各个地点，场景动态变化：


| 时间轴位置 | 环境效果           |
| ----- | -------------- |
| 早期    | 落叶/平原/山地       |
| 中期    | 海洋/雨天/雪天/烈日/夜晚 |
| 后期    | 空中/云层/大气层      |
| 末期    | 星空/太阳系/宇宙      |


场景效果通过背景图像、粒子特效、色彩滤镜等方式实现平滑过渡

## 二、非功能需求

### 1 安全与权限

- 普通用户：仅可浏览与搜索
- 管理员：完整操作权限（通过特殊登录机制验证）

### 2 响应式适配


| 设备  | 时间轴方向  | 布局适配            |
| --- | ------ | --------------- |
| 桌面端 | 水平（默认） | 节点沿水平轴排列        |
| 手机端 | 垂直     | 节点垂直滚动排列，交互逻辑不变 |


### 3 性能要求

- 图片自动压缩，音视频转码，确保加载效率
- 节点保存文件过多时，详情页可正常滚动（修复滑动bug）

### 4 视觉风格

- **基础色调**：深蓝/深紫（暗色调基底）
- **强调色**：粉紫/亮红/霓虹色彩
- **背景质感**：科技感纹理，未来废墟/科技场景融合
- **点缀元素**：赛博朋克建筑轮廓剪影、霓虹光效

## 四、待修复问题清单

### P0 (紧急)

- 音频文件上传后无法正常预览 
- 节点保存文件过多时，详情页无法下滑

### P1 (高优先级)

- 取消/关闭编辑页时，新上传的文件未清理 
- 上传文件后，upload文件夹出现两份同名文件 
- 编辑节点保存后，编辑窗口未自动关闭

### P2 (中优先级)

- 删除节点后，关联的多媒体文件未自动清理
